> Este projeto é um Scraping utilizando Puppeteer 5.5 que tem o objetivo de listar todos os imóveis da Caixa Econômica Federal disponíveis para leilão, definidos por estado e cidade.

## Install

```sh
npm install
```

## Usage

```sh
npm start
```
